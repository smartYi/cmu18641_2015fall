**Attention: This source code is based on java SE-1.8. You may not run this code if your java version is lower than 1.8.

1: What is the source code for?

This is the folder of project-unit1 for the course 18641--Java Smart Phone Development.

2: Important subfolders and documents

When you unzip this folder, you will see following subfolders and documents;

yiq_Project1(folder)
          1)bin(folder)
          2)src(folder)
          3)ZTW.txt(Input data)
          4)ZTW.ser(Serialization data)

Class_relationship_diagram.png
Readme.txt
test_output.txt

yiq_Project1 is the java projects; test_output.txt file is the test results of my experiments for this project;
Assignment_Class_diagram is the class relationship drawn by hand before coding. (Of course some revisements are done during the coding process)
The final document is the Readme.txt file which is exactly the one you are reading.

3: How to run the source code

When you try to run the source code in the Eclipse (version 1.8 is needed), two steps are involved;

1): You should import the whole java project, that is yiq_Project1 into the Eclipse;
2): Then you find the driver package and locate the test.java file and run this file, you can get the accordingly result in the console.
