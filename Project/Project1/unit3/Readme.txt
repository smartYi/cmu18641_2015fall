When you unzip this folder, you will see following subfolders and documents;

1)yiq_project1_unit3
2)Readme.txt
3)Documentation.pdf
4)Class_relationship_diagram.png
5)test_output.txt

yiq_project1_unit3 is the java projects; test_output.txt file is the test results of my experiments for this project;
Class_relationship_diagram is the class relationship;

When you try to run the source code in the Eclipse (version 1.8 is needed), two steps are involved;

1): You should import the whole java project;
2): Then you find the driver package and locate the driver.java file and run this file, you can get the accordingly result in the console.
