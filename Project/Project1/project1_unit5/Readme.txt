1: What is the source code for?

This is the folder of project1-unit5 for the course 18641--Java Smart Phone Development.

2: Important subfolders and documents

When you unzip this folder, you will see following subfolders and documents;

1)project1_unit5_Server
2)project1_unit5_servlet
3)Documentation.pdf
4)server_Class_relationship_diagram.png
5)servlet_Class_relationship_diagram.png
6)Server_test_output.txt
7)Servlet_Test_output.txt
8)Readme.txt

project1_unit5_Server and project1_unit5_servlet are the two projects in this unit; 
Server_test_output.txt file is the test results of server part for this project;
Servlet_Test_output.txt is the test results of servlet part for this project;
server_Class_relationship_diagram.png is the server class relationship;
servlet_Class_relationship_diagram.png is the servlet class relationship;
Documentation.pdf provides an overall view of this project and what is this source code for;
The final document is the Readme.txt file which is exactly the one you are reading.

3: How to run the source code

When you try to run the source code in the Eclipse (version 1.8 is needed), two steps are involved;

1): You should import both project1_unit5_Server and project1_unit5_servlet project;
2): Then you find the Client.java in package client of project1_unit5_servlet project and Server.java in package server of project1_unit5_Server project;
3): Run Server.java firstly and then run Client.java
4): When you finish uploading models then test configure car in website. Run the index.html in WebContent folder on server. Then you can see details in my documentation.
